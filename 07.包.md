## init方法

1、在main被执行前，所有依赖的package的init方法都会被执行

2、不同包的init函数按照包导入的依赖关系决定执行顺序

3、每个包可以有多个init函数

4、包的每个源文件也可以有多个init函数，这点比较特殊

```go
package main

import "fmt"

func init() {
	fmt.Println(123)
}

func init() {
	fmt.Println(456)
}

func main() {
	fmt.Println("789")
}
```
